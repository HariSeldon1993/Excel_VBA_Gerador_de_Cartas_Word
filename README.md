# Excel_VBA_Dynamic_Word_Document_Generator

Projeto desenvolvido em Excel VBA para automatizar a criação de documentos Microsoft Word a partir de dados cadastrados em planilhas Excel.

## Funcionalidades

- Geração automática de documentos Word
- Criação do documento completamente via código
- Leitura de dados da planilha Excel
- Inserção dinâmica de informações do cliente
- Geração automática da lista de produtos
- Padronização de documentos
- Automatização de tarefas repetitivas

## Tecnologias

- Microsoft Excel
- VBA
- Microsoft Word Object Library
- Microsoft Office

## Como executar

1. Baixe o arquivo **Gera_Cartas.xlsm**
2. Abra o arquivo no Microsoft Excel
3. Habilite as macros
4. Informe o Nome de Usuário do computador (quando solicitado)
5. Clique no botão **Gerar Cartas**
6. O documento Word será criado automaticamente

## Observações

Este projeto foi desenvolvido para demonstrar técnicas de automação entre Microsoft Excel e Microsoft Word utilizando VBA.

Todos os dados utilizados são fictícios e destinados exclusivamente para demonstração.
